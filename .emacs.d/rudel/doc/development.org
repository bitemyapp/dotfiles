* Tricks
** Simulating a Slow Network Connection
   /Note: Probably only works for Linux/
   + Installing ::
     #+BEGIN_SRC sh
     tc qdisc add dev lo root handle 1:0 netem delay 200msec
     #+END_SRC
   + Resetting ::
     #+BEGIN_SRC sh
     tc qdisc del dev lo root handle 1:0
     #+END_SRC
